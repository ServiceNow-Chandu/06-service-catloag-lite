# Service Catalog Lite – ServiceNow

A scoped ServiceNow application built using Studio that demonstrates
end-to-end Service Catalog design and automation using Flow Designer.

This project includes multiple catalog items with approvals, task
orchestration, and notifications following enterprise best practices.

---

## Project Objective
To design user-friendly Service Catalog items that automate common IT
requests while enforcing approvals, security controls, and fulfillment
workflows.

---

## Catalog Items Implemented
1. Laptop Request
2. VPN Access Request
3. Application Access Request

---

## Features Implemented
- Catalog Items with structured variables
- Manager and group-based approvals
- Conditional approval logic
- Parallel task creation using For Each
- Automated fulfillment and closure
- Flow Designer–based orchestration
- GitHub source control integration

---

## Technologies Used
- ServiceNow Studio (Scoped App)
- Service Catalog
- Flow Designer
- Approvals
- Catalog Tasks
- Notifications

---

## Demo Scenarios
1. Laptop Request → Manager approval → Hardware task created
2. VPN Request → Manager + Security approval → VPN task created
3. Application Access → Manager approval → multiple tasks created in parallel

---

## Notes
- Built using modern Flow Designer (no legacy workflows)
- Designed for enterprise ITSM use cases
